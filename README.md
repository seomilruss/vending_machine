# vending_machine / 자판기
Toy Project(all Vending machine) / 장난감 프로젝트(자판기)

This project has "one theme, a vending machine." / 본 프로젝트는 "자판기 라는 1가지 주제"를 가지고

1. What can you think of if you give a topic? -> Planning ability / 주제를 주면 어떤 생각을 가능한가? -> 기획 능력
2. How can you speak several languages the same? -> Expressiveness / 몇 가지 언어들을 동일하게 어떻게 구사할 수 있는가? -> 표현력
3. How can you express each language? -> Outline of each language / 각 언어를 어떻게 표현이 가능한가?  -> 각 언어의 윤곽

It shows the above contents. / 상기의 내용들을 보여줍니다.

프로젝트 특징
1. 1개의 언어가 객체를 사용하면 다수의 언어도 동일하게 객체를 사용함(Ex_C에서 구조체 공용체 사용 -> Java, JSP, PHP, Python 등에서 객체 사용)
/  If one language uses an object, many languages use the same object (Ex_C uses a structure community -> Java, JSP, PHP, Python, etc.)
2. 모든 언어는 동일하게 결과가 출력 되어야함 / All languages should have the same output.
