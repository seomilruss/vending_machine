# 자판기
장난감 프로젝트(자판기)
본 프로젝트는 "자판기 라는 1가지 주제"를 가지고

1. 주제를 주면 어떤 생각을 가능한가? -> 기획 능력
2. 몇 가지 언어들을 어떻게 구사할 수 있는가? -> 표현력 
3. 각 언어를 어떻게 표현이 가능한가?  -> 각 언어의 윤곽
